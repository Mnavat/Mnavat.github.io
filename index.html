Web page prototype created with the Angular framework in collaboration with André Nava, this project uses HTML, CSS and TypeScript. It implements some angular directives such as ngModel (for two-way bingind), ngForm, ngClass, etc. Additionally, the angular routing and httpClient libraries are integrated to make the components network and requests to a NoSQL database created in MongoDB through a backend API with the help of the express and mongoose libraries.

Note: The rendering of this page is intended only to be displayed in a web browser on a computer. The web page added to this portfolio shows only the interface, the connection to the database is not currently integrated (for more details of the back-end API check the repository on GitHub).
